# Rules Index

Rules pequenas para consulta rapida durante evolucao do projeto.

## Arquivos

- `architecture.rules.md`: dependencias entre camadas e limites arquiteturais.
- `feature-evolution.rules.md`: como planejar e entregar novas features.
- `cli.rules.md`: compatibilidade e clareza da interface CLI.
- `formatters-writers.rules.md`: separacao entre formatacao e escrita.
- `testing.rules.md`: testes rapidos sem Whisper real.
- `documentation.rules.md`: quando e como atualizar documentacao.

## Uso Recomendado

Use uma rule especifica quando a mudanca tocar aquele tema. Para contexto completo, consulte tambem:

- `../ARCHITECTURE_GUIDELINES.MD`
- `../PROJECT_GUIDELINES.MD`
- `../EVOLUTION_RULES.MD`
