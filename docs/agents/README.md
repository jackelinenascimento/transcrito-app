# Agents Index

Agentes pequenos para orientar evolucoes do `transcrito-app` por responsabilidade.

Estes arquivos estao em `docs/agents/` porque o diretorio `.agents/` esta montado como read-only neste ambiente.

## Agentes

- `architecture-reviewer.md`: revisa limites de Clean Architecture.
- `feature-planner.md`: quebra features em fatias pequenas.
- `cli-evolver.md`: evolui flags, mensagens e compatibilidade da CLI.
- `formatter-builder.md`: cria e revisa formatos de saida.
- `transcription-engine-adapter.md`: adiciona ou revisa engines de transcricao.
- `test-writer.md`: cria testes rapidos e sem Whisper real.
- `docs-maintainer.md`: sincroniza README, guidelines, rules e mapas.

## Uso Recomendado

Escolha um agente pelo tipo de tarefa. Cada arquivo descreve missao, quando usar, contexto obrigatorio, checklist e saida esperada.
